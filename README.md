## Stageopdracht

Setup backend
=
To install composer:
> composer install
> Not found error? > delete vendor folder

To setup the database:
> php artisan migrate
>
> 'yes' to continue
>
> php artisan migrate:fresh --seed

To run the backend:
> php artisan serve

Setup frontend
=
To run the frontend:
> npm i
>
> npm run dev
